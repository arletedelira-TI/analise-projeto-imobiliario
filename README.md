# 🏠 Análise de Dados do Mercado Imobiliário com Pandas

Projeto desenvolvido durante o curso **Pandas: Conhecendo a Biblioteca**, da Alura.

O objetivo deste projeto foi aplicar técnicas de manipulação, limpeza e análise exploratória de dados utilizando a biblioteca **Pandas**, simulando um cenário real de análise de imóveis para aluguel.

---

## 📖 Sobre o projeto

Neste projeto foi realizada uma análise de uma base de dados imobiliária utilizando Python e Pandas.

Ao longo do desenvolvimento foram aplicados conceitos fundamentais de análise de dados, desde a importação da base até a criação de novas variáveis e tratamento das informações.

O projeto demonstra como transformar dados brutos em informações úteis para apoiar tomadas de decisão.

---

## 🎯 Objetivos

- Importar bases de dados utilizando Pandas
- Explorar a estrutura do conjunto de dados
- Realizar análises exploratórias
- Tratar valores ausentes
- Filtrar registros
- Criar novas colunas
- Manipular dados categóricos
- Exportar dados tratados

## 📊 Etapas do projeto

### 1. Conhecendo a base de dados

- Importação dos dados
- Estrutura do DataFrame
- Tipos de dados
- Estatísticas descritivas

---

### 2. Análise exploratória

Foram realizadas análises como:

- Valor médio dos imóveis
- Agrupamentos utilizando `groupby()`
- Consultas utilizando `query()`
- Seleção de informações específicas
- Análise por categorias de imóveis

---

### 3. Tratamento dos dados

Nesta etapa foram aplicadas técnicas de limpeza dos dados:

- Tratamento de valores nulos
- Remoção de registros inconsistentes
- Aplicação de filtros
- Seleção de subconjuntos de dados
- Exportação dos dados tratados

---

### 4. Manipulação dos dados

Foram realizadas transformações como:

- Criação de colunas numéricas
- Operações entre colunas
- Criação de colunas categóricas
- Criação de variáveis binárias
- Engenharia de atributos simples
